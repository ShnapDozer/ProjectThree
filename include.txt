1.SFML:
   1.Все конфигурации, Все платформы:
      C/C++ => общие => дополнительные каталоги вкючаемых файлов.         
      Прописываем путь до папки include (вообще пох какая битность)

   2.Release, Все платформы:
      Компоновщик => Ввод => Дополнительные зависимости.
      Дописываем туда это:
      sfml-graphics.lib
      sfml-window.lib
      sfml-system.lib
      sfml-audio.lib
      sfml-network.lib

   3.Debug, Все платформы:
      Компоновщик => Ввод => Дополнительные зависимости.
      Дописываем туда это:
      sfml-graphics-d.lib
      sfml-window-d.lib
      sfml-system-d.lib
      sfml-audio-d.lib
      sfml-network-d.lib

   4.Все конфигурации, Платформа Win32:
      Компоновщик => общие => дополнительные каталоги библиотек.
      Прописываем путь до папки lib (SFML x32/86)

   5.Все конфигурации, Платформа Win64:
      Компоновщик => общие => дополнительные каталоги библиотек.
      Прописываем путь до папки lib (SFML x64)

   6. После сборки всех вариантов, кидаем бинарники из папок bin по своим местам

2.ImGui:
   1.Project => "VC++ Directories" => "Include Directories"
      устанавливаем путь до папок  ...\imgui-master и ...\imgui-sfml-master  только для 32бит
   2.Debug, Все платформы:
      Компоновщик => Ввод => Дополнительные зависимости.
         Дописываем туда это:
         "opengl32.lib"
         "glu32.lib"
   3.Кидаем все .cpp файлы в проект.
